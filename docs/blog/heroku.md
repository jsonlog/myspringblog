# heroku
Heroku平台与其他部署平台不同之处在于它在虚拟化平台上运行的文件系统是临时的
```
heroku command
heroku login
heroku create //app
heroku apps:rename newname ( --app oldname)
git push heroku master
heroku open //com
heroku logs --tail
heroku ps
heroku domains:add www.example.com
heroku run bash
```
